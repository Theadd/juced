Introduction

This is an unofficial WYSIWYG JUCE component class designed to allow the users design a Graphical User Interface of JUCE components. This component is ready to be embedded into your RAD application within a few steps and allows the export/import of XML files containing the component's structure as well as the creation of *.h/*.cpp containing the generated code for them.

Installation

Step 1: Copy the code somewhere into your project.

Step 2: Place attributes.xml and enumerations.xml into your application working directory.

Step 3: Add #include "***SOMEWHERE***./JUCE_Designer.h" and type in this code:

juced = new JUCE_Designer();

addAndMakeVisible(juced); juced->setBounds(this->getLocalBounds()); juced->setAlwaysOnTop(true);

Create a new Toolbox: params(itemsPerRow, itemSize, itemPadding); Component* toolbox = juced->createToolbox(1, 32, 5);

Add items to our toolbox (more than one toolbox can be created). juced->addToolboxItem(toolbox, "juced_Window", "Window", BinaryData::window_png, BinaryData::window_pngSize); juced->addToolboxItem(toolbox, "juced_Label", "Text label", BinaryData::label_png, BinaryData::label_pngSize); juced->addToolboxItem(toolbox, "juced_TextButton", "Text button", BinaryData::button_png, BinaryData::button_pngSize);

toolbox->setAlwaysOnTop(true); juced->addAndMakeVisible(toolbox, 11);

Viewport *propertyView = juced->getPropertyView(); addAndMakeVisible(propertyView); propertyView->setBounds(100, 100, 200, 400); propertyView->setOpaque(true); propertyView->addToDesktop((int)(ComponentPeer::windowHasTitleBar | ComponentPeer::windowIsResizable));
